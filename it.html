IT Jobs

Help create, process and store electronic data such as text and media. Help test application software.

Information Security Analyst

Plan and execute security measures to protect computer networks. Other jobs include monitoring networks and conducting penetration tests.

Average base salary: $88,000

Required Knowledge: A degree in Information Systems, Computer Science, or another relevant field.

Business Analyst

Learn and understand how technologies can help businesses. Use data to help bridge the gap between business and technology to meet customer needs. Provide requirements to software developers to create a piece of software to help the business.

Average base salary: $70,000

Required Knowledge: A good understanding of technologies such as databases, networks, and software applications. Good communication skills and knowledge of business structure and model. A degree in Computer Science, Business, or Business Analysis.

Database Administrator

Handle the storage and organization of data for a business. Maintain and update database schemas. Perform routine backups and maintenance.

Average base salary: $74,000

Required Knowledge: A degree in Computer Science. Knowledge in Database systems such as MySQL, Oracle, Microsoft SQL server, etc.

Software Tester

Create manual and automatic tests to make sure software is free of bugs and is ready for production. Report bugs to software developers.

Average base salary: $60,000

Required Knowledge: A degree in Computer Science and knowledge in creating tests with langauges such as Python and others.
