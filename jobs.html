Popular Jobs

Some of the popular jobs in the tech field.

Artificial Intelligence/Machine Learning Engineer

Use data to train models to solve problems in areas such as image recognition, natural language processing, and others!

Average Base Salary: $146,000

Required Knowledge: Statistics, Linear Algebra, Python and a good background in Computer Science. Libraries such as Tensorflow, Pytorch and others.

Data Scientist

Compile, clean and present data in a nice organized way in order to make decisions.

Average Base Salary: $95,000

Required Knowledge: Statistics, Linear Algebra, a strong background in Python data science libraries such as NumPy, Pandas and Matplotlib.

Web Developer

Use frameworks and programming languages to create and maintain powerful websites and/or web applications.

Average Base Salary: $80,000

Required Knowledge: HTML, CSS, JavaScript, and a stack such as LAMP, MERN, etc.

App Developer

Use popular programming langauges to create mobile applications and games for iOS and Android. This is a highly sought-after job as the popularity of mobile devices increases.

Average base salary: $70,000

Required Knowledge: App development languages such as Swift, Java, Kotlin. Frameworks such as Flutter and React Native are also recommended.
