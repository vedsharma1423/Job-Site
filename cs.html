Computer Science Jobs

Design, write, and test software for any kind of devices to solve any kind of problem.

Software Developer

Create software applications using programming languages and solve problems. Developers work on all aspects of a project from design and development to testing and production. This is a fast growing occupation.

Average base salary: $107,000

Required Knowledge: Programming language such as Java, C++, Python, etc. Good understanding of data stuctures and algorithms. Degree in Computer Science

Computer Research Scientist

They invent technology to solve problems in different areas. They also find uses for existing technology to solve such problems. They also analyze patterns in large datasets. They work on programs that control robots

Average base salary: $123,000

Required Knowledge: A degree in Computer Science or information systems. Good knowledge of programming languages such as Python, Java, C++, etc.

Computer Systems Manager

Oversee the implementation of technology in a company. Evaluate software and hardware technologies for purchasing and development.

Average base salary: $146,000

Required Knowledge: Good communication skills and knowledge in software and hardware technologies and components. A degree in computer science, computer engineering or something similar is required.
